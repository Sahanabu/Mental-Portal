- Auto-applied to all API requests via axios interceptor

## Usage

### For Users
1. Click language selector in navbar
2. Choose: English / हिंदी / ಕನ್ನಡ
3. All UI text and AI responses change immediately
4. Preference saved automatically

### For Developers
```javascript
// Use translations in components
import { useLanguage } from '@/contexts/LanguageContext';

function MyComponent() {
  const { t, language } = useLanguage();
  
  return <h1>{t?.app?.title || 'Fallback Text'}</h1>;
}
```

## API Integration

### Automatic (Recommended)
Language is auto-added to all requests via interceptor:
```javascript
const response = await chatAPI.sendMessage(message);
// Language automatically included
```

### Manual (Optional)
```javascript
const response = await chatAPI.sendMessage(message, sessionId, 'hi');
```

## Testing

### Test Language Switching
1. Start application: `npm run dev`
2. Open http://localhost:3000
3. Click language selector in navbar
4. Switch between English, Hindi, Kannada
5. Verify UI text changes
6. Test chat - AI should respond in selected language
7. Complete assessment - analysis should be in selected language

### Test API Directly
```bash
# Hindi chat
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"message": "मुझे चिंता हो रही है", "language": "hi"}'

# Kannada chat
curl -X POST http://localhost:5000/api/chat \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer YOUR_TOKEN" \
  -d '{"message": "ನನಗೆ ಆತಂಕವಾಗಿದೆ", "language": "kn"}'
```
